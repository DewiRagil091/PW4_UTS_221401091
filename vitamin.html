Vitamin Boost

Selamat datang di halaman vitamin! Berikut adalah vitamin yang dapat memberikan kekuatan pada tubuh Anda:

Vitamin C

Vitamin C membantu meningkatkan sistem kekebalan tubuh. Sumber alami vitamin C meliputi jeruk, stroberi, dan brokoli.

Vitamin D

Vitamin D baik untuk kesehatan tulang. Anda dapat mendapatkan vitamin D dari sinar matahari, ikan salmon, dan telur.

Vitamin A

Vitamin A penting untuk kesehatan mata. Makanan seperti wortel, bayam, dan ubi jalar kaya akan vitamin A.

Vitamin B

Vitamin B kompleks mendukung energi dan metabolisme. Sumbernya termasuk daging, kacang-kacangan, dan biji-bijian.
